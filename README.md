## ✨ What's New in v3.0?

### 🎨 Premium "Deep Forest" Design
Experience a stunning new visual identity. We've replaced legacy layouts with a high-fidelity design system inspired by deep forest aesthetics.
*   **Modern UI**: Fully built with Material Design 3 components.
*   **Dynamic Layouts**: Smooth, card-based interface with refined elevation and shadows.

### 🌙 True Dark Mode
A first-class dark theme experience with optimized contrast and forest-green accents, designed to be easy on the eyes while looking premium.

### ⚙️ Modernized Core
*   **Android 15 Ready**: Targeted for SDK 35 with support for current security and performance standards.
*   **AndroidX Architecture**: Migrated from legacy support libraries to modern AndroidX for better stability.
*   **Enhanced UX**: Redesigned setup/tutorial flow with real-time permission status feedback.

## 🚀 Getting Started

1.  **Download**: Get the latest [SPAMit_v3.0.apk](exports/SPAMit_v3.0.apk) from the exports folder or the Releases page.
2.  **Permissions**:
    *   **Accessibility Service**: Required to automate typing and sending messages.
    *   **Overlay Permission**: Required to show the floating control widget.
3.  **Setup**: Open WhatsApp and ensure your "Enter" key is set to send messages for the best experience.
